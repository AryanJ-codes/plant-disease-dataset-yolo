# Intelligent Crop Disease Identification and Decision Support Framework

This project presents an intelligent, data-driven framework for detecting and classifying crop diseases from field images and providing actionable treatment recommendations.

The system integrates high-performance computer vision models with AI-driven decision support to assist early disease identification and farm-level decision-making.
